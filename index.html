Projects

Kinematic Modelling of ABB IRB 120 Robotic Arm for Accurate Placement of Ball Grid Array (BGA) on a Printed Circuit Board using Vision

Objective: Implement Pick and Place of BGA(Ball Grid Array) chips onto PCB with the help of ABB IRB 120 Industrial robot.

Tasks Implemented:

1) The location of the BGA chip in the table and its placement on the correct location on the PCB was determined using Vision techniques and the entire processing was implemented in OpenCV- ROS.

2) The trajectory planning and inverse and forward kinematics were implemented using the Eigen Library in ROS-Gazebo.

3) The simulation of the pick and place task was implemented in Gazebo using ROS.

4) A prototype hardware implementation of the entire process was implemented on the Real ABB IRB 120 robot using ROS-Industrial and MoveIt Packages.
